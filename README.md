# ItemStorageBrowser (Item Storage Browser) — Аддон для WoW 3.3.5a

Аддон **ItemStorageBrowser** разработан специально для игроков сервера Sirus x5, проходящих испытание "Один шанс".

Этот инструмент предназначен для улучшенного поиска и подбора предметов, находящихся на складе гильдии **Phoenix Nest**.

⚠️ **Важно:** Использование аддона полезно **только** игрокам, проходящим испытание «Один Шанс» на Sirus x5.

---

## 📦 Описание проекта

**Item Storage Browser** — это модификация (addon), которая позволяет удобно просматривать содержимое склада предметов через внутриигровой интерфейс. 

### Основные функции:
* Поиск по названию и подбор по параметрам предметов, находящихся на складе.
* Отображение количества предметов в подсказке при наведении мышки на предмет, с указанием ника складского персонажа, у которого хранится этот предмет.

### Общие команды:
* /isb - переключение видимости окна
* /isb show - показать окно
* /isb hide - скрыть окно

### Специальные команды:
* /bagfree - отправить в чат свободное место в сумках
* /bankfree - отправить в чат свободное место в личном банке
* /free - отправить в чат общее количество свободного места

## 🚀 Как скачать и установить (Download)

Для получения актуальной версии **ItemStorageBrowser**, GitHub предоставляет несколько способов:

### Способ 1: Прямая загрузка (**рекомендуется**)
1. Перейдите в раздел [Releases](https://github.com/WorldOfSitesNet/ItemStorageBrowser/releases).
2. Скачайте архив `ItemStorageBrowser.zip`.

### Способ 2: Скачивание исходного кода (**НЕ** рекомендуется)
1. Нажмите на зелёную кнопку **Code** вверху страницы.
2. Выберите **Download ZIP**.

## 📥 Инструкция по установке

1. Распакуйте скачаный архив с аддоном.
2. Удалите старую версию папки `ItemStorageBrowser` из папки аддонов клиента игры: `\Interface\AddOns\`
3. Переместите новую папку `ItemStorageBrowser` в ту же папку с аддонами.
4. Убедитесь, что папка, которую вы помещаете в аддоны называется именно `ItemStorageBrowser`, а не  `ItemStorageBrowser-main`,  `ItemStorageBrowser-2.8.0-beta` или еще как-то иначе: если заметили неправильное название - переименуйте папку этого аддона в `ItemStorageBrowser`.
5. Убедитесь, что в папке аддона `ItemStorageBrowser` нет вложенной папки с таким же названием, а структура файлов и папок - такая же, как показано выше: папки Core, Tabs, несколько файлов: если нет, удалите эту папку с аддоном и пройдите по пунктам инструкции по скачиванию и установке, заново.
6. Когда аддон скачан и установлен правильно: зайдите в игру, в меню выбора персонажа нажмите кнопку **Модификации** и убедитесь, что аддон включен.

> **Примечание:** Если аддон горит красным, поставьте галочку "Устаревшие модификации".

## 🔄 Обновление базы данных (ItemStorageDB)

### Автоматическое обновление
Для автоматического обновления БД аддона используется наше приложение [ItemStorage Manager](https://github.com/Phoenix-Guilds/ItemStorageDesktop) 👉 следуйте инструкциям, опубликованным на странице этого приложения для скачивания, установки и использования.

### Ручное обновление (не рекомендуется)
Вы можете обновить данные складского аддона вручную. Свежий файл `ItemStorageDB.lua` публикуется в отдельной ветке данного репозитория `data` некоторыми помощниками по складу [открыть ветку data](https://github.com/Phoenix-Guilds/ItemStorageBrowser/tree/data) 👉 Следуйте инструкциям на странице для скачивания и установки обновленной БД в складской аддон. В будущем - все помощники по складу смогут обновлять БД аддона автоматически.
